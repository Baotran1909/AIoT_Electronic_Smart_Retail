# Raspberry Pi Pico

Raspberry Pi Pico là board vi điều khiển dùng chip RP2040, phù hợp cho học lập trình nhúng, điều khiển cảm biến và các ứng dụng thời gian thực nhẹ.

## Đặc điểm chính
- Sử dụng chip RP2040.
- Logic 3.3V.
- Hỗ trợ MicroPython và C/C++.
- Có ADC, PWM, I2C, SPI, UART.

## Ứng dụng
- Đọc cảm biến.
- Điều khiển servo, LED, buzzer.
- Hệ thống đo lường đơn giản.
- Robot cơ bản.

## Lưu ý kỹ thuật
- Không cấp 5V vào GPIO.
- Nếu cần WiFi nên dùng Pico W hoặc module ngoài.
- Cần nối chung GND khi giao tiếp với module khác.

## Thường dùng với
- OLED SSD1306.
- DHT22.
- Servo SG90.
- Buzzer.
- LDR sensor.

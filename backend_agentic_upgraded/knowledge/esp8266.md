# ESP8266

ESP8266 là vi điều khiển tích hợp WiFi, thường dùng trong các ứng dụng IoT đơn giản như điều khiển thiết bị qua mạng, gửi dữ liệu cảm biến hoặc xây dựng web server nhỏ.

## Đặc điểm chính
- Tích hợp WiFi.
- Logic 3.3V.
- Số lượng GPIO ít hơn ESP32.
- Giá thành rẻ, phù hợp dự án nhỏ.

## Ứng dụng
- Điều khiển relay qua WiFi.
- Gửi dữ liệu DHT11/DHT22 lên server.
- Smart switch.
- Web server điều khiển thiết bị.

## Lưu ý kỹ thuật
- Không chịu được mức logic 5V ở GPIO.
- Cần nguồn 3.3V ổn định, dòng đủ lớn.
- GPIO hạn chế nên cần cân nhắc khi kết nối nhiều module.

## Thường dùng với
- Relay module.
- DHT11/DHT22.
- LDR sensor.
- MQTT/Firebase.

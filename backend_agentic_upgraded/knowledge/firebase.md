# Firebase trong IoT

Firebase Realtime Database thường dùng để đồng bộ dữ liệu giữa thiết bị IoT, backend và giao diện web theo thời gian thực.

## Đặc điểm chính
- Đồng bộ realtime.
- Dữ liệu dạng JSON tree.
- Phù hợp dashboard và trạng thái thiết bị.

## Ứng dụng
- Hiển thị cảm biến realtime.
- Đồng bộ trạng thái thiết bị.
- Quản lý đơn hàng hoặc smart locker.
- Lưu lịch sử hoạt động đơn giản.

## Lưu ý kỹ thuật
- Cần cấu hình security rules phù hợp.
- Không nên lưu dữ liệu quá lớn dạng không tối ưu.
- Thiết bị cần kết nối Internet ổn định.

## Thường dùng với
- ESP32.
- Web dashboard.
- FastAPI backend.
- Smart Retail.
